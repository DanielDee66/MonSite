if

La structure conditionnelle basée sur le seul mot clef "if" est la plus simple. Elle permet d'exécuter une ou pleusieurs instruction(s) si la condition est rencontrée.


if ... else

La structure conditionnelle basée sur les mots clef "if" et "else" est la assez similaire à la structure précédente. Elle permet d'exécuter une ou plusieurs instruction(s) si la condition est rencontrée. Dans le cas contraire, ce sont les instructions qui sont dans le second bloc (sous le "else") qui sont exécutées.


if ... elif ... else

La structure conditionnelle if ... elif ... else permet de texter plusieurs conditions les unes après les autres. Elle peut être remplacée par des "if ... else" imbriqués.
