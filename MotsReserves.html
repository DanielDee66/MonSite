else

Le mot clef "else" peut être utilisé dans plusieurs cas:

  • avec "if" pour construire des structures conditionnelles ;
  • avec "elif" pour construire des structures conditionnelles ;
  • avec "for" et "while" pour exécuter des instructions si la boucle n'a pas été interrompue à l'aide d'une instruction "break" ;
  • avec "try" pour exécuter des instructions s'il n'y a pas d'exception.
